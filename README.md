# Ban Spawn Ender Dragon

[中文](#中文) | [English](#english)

## 中文

### 模组介绍

一个简单的 Fabric 模组，用于阻止末影龙的复活。

### 功能特性

- 🚫 **阻止复活** - 防止玩家在末地复活末影龙
- ⚙️ **可配置** - 支持通过配置文件自定义行为
- 🌐 **多语言** - 支持中文和英文提示消息
- 💬 **消息控制** - 可开关阻止放置时的提示消息
- 🔧 **命令支持** - 提供重载配置和语言切换命令

### 使用方法

#### 安装
1. 确保已安装 Fabric Loader
2. 将模组文件放入 `mods` 文件夹
3. 启动游戏
